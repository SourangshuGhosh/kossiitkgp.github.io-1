Ubuntu Installation Fest




2 October 2018




Ubuntu installation fest by Kharagpur Open Source Society




Event Overview

KOSS conducts Ubuntu Installation Fest every year in the Autumn semester, open to everyone in IIT Kharagpur, for those who would like to get a Linux based system as either Primary or Secondary Operating system alongside windows.

Since, Linux is always helpful for almost all the workshops by KOSS and preferred operating system for programming, hence it was a primary need for it being among the first events of the year.
